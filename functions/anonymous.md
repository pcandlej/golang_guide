# Go语言匿名函数
## 一、定义与核心特性
匿名函数（又称函数字面量）是Go语言中无名称的函数，核心作用是创建内联函数，同时支持形成闭包，具备灵活、轻量的特点，是Go语言中函数式编程的重要基础。

## 二、基本语法
```go
func(parameter_list)(return_type) {
    // 函数体逻辑
    // 若指定返回类型，需通过return语句返回结果；无返回类型则无需return
}() // 直接调用时添加括号，赋值给变量或作为参数/返回值时可省略
```

## 三、主要使用场景
### 1. 直接调用（定义后立即执行）
定义匿名函数后，通过末尾添加`()`直接执行，适用于仅需执行一次的简短逻辑。
```go
func() {
    fmt.Println("直接执行的匿名函数")
}()
```

### 2. 赋值给变量
将匿名函数赋值给变量，变量可作为普通函数调用，适用于需多次执行该逻辑的场景。
```go
func main() {
    fn := func() {
        fmt.Println("赋值给变量的匿名函数")
    }
    fn() // 调用匿名函数
}
```

### 3. 接受参数
匿名函数支持定义参数列表，调用时传入对应参数，实现动态传值。
```go
func(ele string) {
    fmt.Println(ele)
}("测试参数") // 传入参数并执行
```

### 4. 作为参数传递给其他函数
可将匿名函数作为实参传入另一个函数，目标函数需将函数类型声明为参数类型，适用于回调函数等场景。
```go
// 接收函数类型参数的函数
func process(fn func(a, b string) string) {
    fmt.Println(fn("Hello", "World"))
}

func main() {
    // 匿名函数作为参数传入
    process(func(a, b string) string {
        return a + " " + b
    })
}
```

### 5. 作为返回值从其他函数返回
函数可将匿名函数作为返回值，调用该函数后会得到匿名函数实例，后续可按需调用，适用于封装逻辑、延迟执行等场景。
```go
// 返回匿名函数的函数
func getFunc() func(x, y string) string {
    return func(x, y string) string {
        return x + "-" + y
    }
}

func main() {
    fn := getFunc()
    fmt.Println(fn("Go", "Lang")) // 调用返回的匿名函数
}
```

## 四、核心优势
匿名函数兼具灵活性与多功能性，可快速实现内联逻辑、回调机制、闭包封装等需求，无需单独定义命名函数，能简化代码结构，提升开发效率，是Go语言中增强编程灵活性的重要工具。
