# golang_guide

Go，也作 Golang，是一种过程式静态类型编程语言，其语法与 C 语言类似。它由 Google 的 Robert Griesemer、Rob Pike 和 Ken Thompson 于 2007 年开发，并于 2009 年作为开源编程语言发布，主要用于 Google 的生产系统。Golang 是开发者中最流行的编程语言之一。它提供了丰富的标准库、垃圾回收和动态类型功能。
